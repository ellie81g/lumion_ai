Quick guide 

1. Open projectr in any code redactor
2. Open bash terminal
3. Create virtual env

```
python -m venv venv
```
4. Open your virtual env
```
.\venv\Scripts\activate
```
5. Install dependencies
```
pip install -r requirements.txt
```
6. Run the project
```
python app.py
```
